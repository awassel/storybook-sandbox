# Rightpoint FED/UI CMS Boilerplate

This FED/UI boilerplate is a standard front-end template for building componentized, robust, and adaptable web apps or sites within the Rightpoint CMS practice. It utilitzes [**Storybook's HTML**](https://storybook.js.org/docs/html/get-started/introduction) framework for building UIs with a [**component-driven**](https://componentdriven.org) process starting with atomic components and ending with pages.

<div class="subheading">Build Package</div>

This project has a set of build scripts that bundles all assets within the `./dist` directory.

- `css` contains a compiled, autoprefixed version of all Sass files into a single bundle - both minified and unminified.
- `es` contains transpiled JavaScript files into ES6 modules.
- `html` is a version of all component HTML files, copied into a specific directory.
- `scripts` contains a copiled version of all JavaScript files into a single bundle - both minified and unminified.
- `storybook` contains an optional build directory for a static version of this Storybook project.
- `umd` contains transpiled JavaScript files into UMD format.
